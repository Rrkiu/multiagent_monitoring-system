# 장비 오사용 대응 가이드

## 심각도
**HIGH** - 긴급 조치 필요

## 즉시 조치사항
1. 장비 즉시 정지 지시
2. 작업자 안전 확인
3. 장비 오사용 내역 확인
4. 올바른 사용법 재교육

## 예방 조치
1. 장비 사용 전 교육 의무화
2. 장비별 작업 매뉴얼 비치
3. 자격증 소지자만 사용 가능하도록 제한
4. 정기적인 장비 점검

## 재발 방지
- 장비 사용 자격 관리 강화
- 부적절한 사용 시 즉시 사용 금지
- 장비 안전 점검 체크리스트 운영
- 위험 장비는 별도 보관 및 관리

## 관련 법규
- 산업안전보건법 제34조 (기계 안전)
- 산업안전보건기준에 관한 규칙 제3장
