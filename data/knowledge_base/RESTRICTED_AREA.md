# 제한구역 무단 출입 대응 가이드

## 심각도
**MEDIUM** - 즉시 조치 필요

## 즉시 조치사항
1. 작업자에게 즉시 퇴거 지시
2. 출입 사유 확인
3. 권한 확인 (출입 허가증 등)
4. 무단 출입인 경우 경고 조치

## 예방 조치
1. 제한구역 표시 강화 (펜스, 경고판)
2. 출입 통제 시스템 설치
3. 권한별 출입 관리 체계 구축
4. 정기적인 보안 교육

## 재발 방지
- CCTV 모니터링 강화
- 출입 기록 관리 시스템 도입
- 제한구역 접근 권한 재검토
- 보안 인력 배치

## 관련 법규
- 산업안전보건법 제36조 (출입 제한)
- 개별 사업장 안전 수칙
